A web-based application that allows **patients to make appointments** with a GP clinic and **staff to manage bookings and patient information**.  
This project was created as part of my learning in web development and software development principles.

---

## Features
- Patients can:
  - Register and log in
  - View available appointment slots
  - Book appointments
  - Cancel or reschedule appointments

- Staff can:
  - Log in to a secure staff dashboard
  - View, approve, or cancel patient appointments
  - Manage patient information
  - Track appointment history

---

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript  
- **Backend / Server-side:** ( Python / PHP / Node.js )  
- **Database:** SQL /  MySQL  
- **Tools:** VS Code
